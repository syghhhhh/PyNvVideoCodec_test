
## 3. 详细实现步骤

### 3.1 步骤一：新增 GPU NV12 帧封装类

**文件**：`utils/data_prepare.py`
**位置**：文件末尾

```python
# ======================== PyNvVideoCodec GPU 编码支持 ========================

class GpuNV12Frame:
    """
    GPU 上的 NV12 帧封装类，用于 PyNvVideoCodec GPU buffer 模式编码。

    NV12 内存布局：
      - Y  平面: height × width (亮度，每个像素一个字节)
      - UV 平面: (height/2) × width (色度，U/V 交错排列)

    注意：Y 和 UV 必须位于同一块连续 GPU 内存中，
    编码器内部通过 Y 指针 + pitch*height 偏移来定位 UV 数据。
    """

    def __init__(self, rgba_tensor):
        """
        从 RGBA tensor 创建 NV12 帧

        :param rgba_tensor: (1, 4, H, W) RGBA tensor，值范围 [0, 1]，位于 GPU
        """
        import cupy as cp

        # 提取 RGB 通道 (值范围 [0, 255])
        r = (rgba_tensor[0, 0, :, :] * 255).to(torch.uint8)
        g = (rgba_tensor[0, 1, :, :] * 255).to(torch.uint8)
        b = (rgba_tensor[0, 2, :, :] * 255).to(torch.uint8)

        height, width = r.shape

        # RGB → YUV (BT.601 标准)
        # Y = 0.299*R + 0.587*G + 0.114*B
        # U = -0.14713*R - 0.28886*G + 0.436*B + 128
        # V = 0.615*R - 0.51499*G - 0.10001*B + 128
        y = (0.299 * r + 0.587 * g + 0.114 * b).to(torch.uint8)
        u = (-0.14713 * r - 0.28886 * g + 0.436 * b + 128).clamp(0, 255).to(torch.uint8)
        v = (0.615 * r - 0.51499 * g - 0.10001 * b + 128).clamp(0, 255).to(torch.uint8)

        # 转换为 cupy array (用于 CUDA Array Interface)
        y_cp = cp.asarray(y)
        u_cp = cp.asarray(u)
        v_cp = cp.asarray(v)

        # 分配连续 GPU 内存
        y_size = height * width
        uv_size = (height // 2) * width
        self._buf = cp.empty(y_size + uv_size, dtype=cp.uint8)

        # 填充 Y 平面
        self._buf[:y_size] = y_cp.flatten()

        # 填充 UV 平面 (U/V 交错，2x2 下采样)
        # UV 平面布局: 每 2 个相邻像素为 (U, V) 对
        uv = self._buf[y_size:].reshape(height // 2, width)
        # U 分量: 取 2x2 块左上角像素的 U 值
        uv[:, 0::2] = u_cp[::2, ::2]
        # V 分量: 取 2x2 块左上角像素的 V 值
        uv[:, 1::2] = v_cp[::2, ::2]

        # 创建视图 (view)，不拷贝数据，指针仍指向同一块内存
        self.y_plane = self._buf[:y_size].reshape(height, width, 1)
        self.uv_plane = self._buf[y_size:].reshape(height // 2, width // 2, 2)

    def cuda(self):
        """
        返回 CUDA Array Interface，供 PyNvVideoCodec 编码器使用

        :return: [y_plane, uv_plane] 列表，每个元素都实现了 CUDA Array Interface
        """
        return [self.y_plane, self.uv_plane]
```

## 7. 风险与注意事项

### 7.1 内存管理
- `GpuNV12Frame` 每帧创建新对象，依赖 Python GC 回收
- 对于长视频，确保 GPU 内存充足

### 7.2 编码参数
- 当前使用默认编码参数
- 可根据需要调整 `bitrate`、`preset` 等参数

### 7.3 兼容性
- 仅支持 NVIDIA GPU
- 需要安装 CUDA 工具包

## 8. 后续优化建议

1. **批量编码**：可以考虑预分配 GPU 内存池，减少内存分配开销
2. **编码参数调优**：根据实际需求调整 `preset`（p1-p7）平衡速度和质量
3. **多流并行编码**：对于多任务场景，可利用 NVENC 支持多会话的特性
